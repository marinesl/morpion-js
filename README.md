# Project
A Tic Tac Toe with Node (+ Express + EJS)

Data saved in a JSON file. Example:
```json
[
    {
        "id": 0,
        "board": [0,0,0,0,0,0,0,0,0],
        "player1": "Toto",
        "player2": "Titi",
        "current": "player1",
        "winner": null
    }
]
```
